POSIX_ME_HARDER is a bourne-like shell module for improving script portability. It attempts to normalize shell settings and ensure POSIX behaviour in supportd shells.

The name POSIX_ME_HARDER comes from a short-lived environment variable that could be used to toggle standards-compliant behaviour.

What's in the box?
- POSIX "sh" emulation for various shell environments
- Normalized values for POSIX-defined environment variables
- Bugfixes related to non-compliant shell behaviour
